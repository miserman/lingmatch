lma_termcatR Documentation

Document-Term Matrix Categorization

Description

Reduces the dimentions of a document-term matrix by dictionary-based categorization.

Usage

lma_termcat(dtm, dict, term.filter = NULL)

Arguments

dtm

A matrix with words as column names.

dict

A list object with named character vectors as word lists.

term.filter

A regular expression string used to format the text of each term (passed to gsub). For example, if terms are part-of-speech tagged (e.g., 'a_DT'), filter='_.*' would remove the tag.
