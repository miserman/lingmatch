download.lsspaceR Documentation

Download Latent Semantic Spaces

Description

Downloads the specified semantic space.

Usage

download.lsspace(space = "default", dir = "~/Latent Semantic Spaces")

Arguments

space

name of the space you wish to download. Options include 'default' and 'tasa'.

dir

desired path to the downloaded space; default is '~/Latent Semantic Spaces'.

Details

Spaces are slightly altered and reprocessed versions of those available at https://sites.google.com/site/fritzgntr/software-resources/semantic_spaces.

Spaces can (sometimes) be downloaded directly from https://www.myweb.ttu.edu/miserman/lsspaces/.
