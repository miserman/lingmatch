download.lsspaceR Documentation

Download Latent Semantic Spaces

Description

Downloads the specified semantic space from the to lingmatch github repository to the lingmatch data folder.

Usage

download.lsspace(space = "default", type = "rda")

Arguments

space

name of the space you wish to download. Options include 'default', 'tasa', and 'blog'.

type

the type of file you wish to download, either 'rda' or 'sqlite'. If you plan on loading the space into memory before use, rdas will be fastest, otherwise use sqlites.

Details

Spaces are slightly altered and reprocessed versions of those available at http://www.lingexp.uni-tuebingen.de/z2/LSAspaces/.

Spaces can be downloaded directly from https://github.com/miserman/lingmatch/tree/master/spaces/.
